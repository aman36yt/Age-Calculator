# 🎂 Age Calculator Web App

<p align="center">
  <b>A simple and responsive age calculator that calculates your exact age using HTML, CSS, and JavaScript 🚀</b>
</p>

---

## ✨ Features

🎂 Calculate exact age (years, months, days)
📅 Input date of birth easily
⚡ Instant calculation
📱 Responsive design
🎨 Clean and user-friendly UI

---

## 🛠️ Tech Stack

* 🧱 HTML5 – Structure
* 🎨 CSS3 – Styling
* ⚡ JavaScript – Calculation logic

---

## ⚙️ How It Works

* User enters their date of birth
* JavaScript calculates the difference between current date and birth date
* Displays age in:

  * Years
  * Months
  * Days
